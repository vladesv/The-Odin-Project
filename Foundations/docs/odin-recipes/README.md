# Odin Recipes

This project is a basic recipe website built as part of **The Odin Project** Foundations course.

The goal of this project is to practice fundamental **HTML** skills by creating a multi-page website with proper structure and links between pages.

## Project Description

The website consists of:
- A main **index page** that links to individual recipes
- Three recipe pages:
  - Lasagna
  - Crepes
  - Rice with Chicken
